Flexbox

Flexbox is used to make a flexible layout to automatically and dynamically alter the items' widths and heights to accoomdate many different device screen sizes. without having to manually position elements. It flexes both horizontally and vertically, as opposed to inline or block displays. To use it you need to define a class for a flex container, where you use display: div, and likely define a class for for items using the arrow with the container and div. (ask mentor how to best word). They are most appropriate for small scale layouts. Large scale layouts should use grid.
